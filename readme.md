## Tentang eXchanger

Exchanger adalah layanan tukar uang dan pembelian uang.


## Data Standart

**Status Transaction**
```
ID  NAME
1   waiting
2   success
```
Data ini nantinya menjelaskan status transaksinya.


**Type Transaction**
```
ID  NAME
1   buy
2   sell
3   return
```
Disini menjelaskan tipe transaksinya seperti apa.

**User Roles**
```
ID  NAME
1   admin
2   user
```
Role ini digunakan untuk user role
